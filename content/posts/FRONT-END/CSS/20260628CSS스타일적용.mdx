## 들어가기전  


CSS코드는 어디에 써도 동작은 하지만 관례상 아래의 순서로 작성

```css
/* 1. 리셋 / 전역 설정 */
* { }
body { }

/* 2. 레이아웃 큰 틀 */
header { }
nav { }
main { }
footer { }

/* 3. 섹션별 세부 스타일 */
#about { }
#skills { }
#contact { }

/* 4. 컴포넌트 (버튼, 폼 등) */
button { }
input { }

/* 5. 반응형 (항상 맨 마지막) */
@media (max-width: 768px) { }

```
---

## *{... }모든 요소 css 적용
```css
/* 기본 리셋*/
/* *{...  }모든 html요소에 한번에 스타일 적용  */
* {
/*요소의 전체 크기에 패딩과 테두리 포함 */
box-sizing: border-box;
/*바깥 여백을 0으로 초기화*/
margin:0;
/*안쪽 여백을 0으로 초기화*/
padding:0;
}
```
---

## body{} 전체 화면의 기본 글꽅 뼈대(틀)을 잡는 역할

```css
/*전체 폰트, 줄간격 설정*/
/*body라고 해서 html의 body가 x*/
body{
    /*글꼭 설정(고딕체 계열의 기본 폰트)*/
    font-family:sans-serif;
    /*줄간격 설정(글자 크기의 1.6배)*/
    line-height:1.6;
    /*글자 색상 설정(#333은 어두운 회색)*/
    color:#333;
}

```

## nav{... }네비게이션 
```css
nav{
    /*내부 요소들을 가로로 나란히 정렬*/
    display : flex;
    /*양 끝 정렬(좌측 로고, 우측 메뉴로 분리)*/
    justify-content: space-between;
    /*내부 요소들을 수직(상하) 중앙 정렬*/
    align-items : center;
    /*안쪽 여백 설정(위 16픽셀, 좌우 40픽셀*/
    padding: 16px 40px;
    /*백그라운드 컬러 어두운 진회색 배경지정*/
    background-color: #2D333B;
    /*글자 색상을 흰색으로 지정*/
    color: white;
    /*스크롤 시 화면에 고정시키는 배치 방식*/
    position : sticky;
    /*화면 맨 위(0)위치에 고정*/
    top : 0 ;
}
```
---
##  nav ul{... } 내비게이션 메뉴 목록 설

```css
nav ul{
    /*메뉴 항목들을 가로로 나란히 정렬*/
    display : flex;
    /*리스트 앞의 기본 점(점 기호 제거)*/
    list-style : none;
    /*메뉴 항목 사이 간격을 24 픽셀로 설정*/
    gap: 24px
}
```

---
## nav a{... } 내비게이션 링크(a)설정

```css
nav a{
    /*링크 글자 색상을 흰색으로 지정*/
    color : white;
    /*링크의 기본 밑줄 제거*/
    text-decoration:none;
}
```
---

## nav a:hover {...} 내비게이션 호버 효과(커서를 어떤요소위에 올렸을 때 나타나는 시각적 반응)
```css
nav a:hover {
  /* 마우스를 올렸을 때 요소를 70%만큼만 보이게 하여 살짝 흐려지는 효과를 줌 */
  opacity: 0.7;

  /* 투명도가 변할 때 0.2초 동안 애니메이션처럼 부드럽게 전환되도록 설정 */
  transition: opacity 0.2s;
}
```

## 본문 설정


```css
/* 섹션 공통 */
section {
  /* 안쪽 여백 설정 (위아래 80px, 좌우 40px) */
  padding: 80px 40px;

  /* 최대 가로 폭을 800px로 제한 (콘텐츠가 너무 퍼지지 않게 함) */
  max-width: 800px;

  /* 위아래 여백은 0, 좌우 여백은 자동으로 설정하여 박스를 화면 가운데에 정렬 */
  margin: 0 auto;
}

/* about 섹션 */
#about h1 {
  /* 제목 글자 크기를 48px로 설정 */
  font-size: 48px;
  /* 제목 아래쪽에 16px의 바깥 여백을 주어 본문과 간격을 띄움 */
  margin-bottom: 16px;
}

#about p {
  /* 본문 글자 크기를 18px로 설정 */
  font-size: 18px;
  /* 글자 색상을 body(#333)보다 살짝 더 밝은 회색으로 설정 */
  color: #555;
}

/* skills 섹션 */
#skills {
  /* 배경색을 아주 밝은 회색으로 설정하여 다른 섹션과 시각적으로 구분 */
  background-color: #F8F9FA;
}

#skills h2 {
  /* 부제목 글자 크기를 24px로 설정 */
  font-size: 24px;
  /* 부제목 글자 색상을 nav 바와 같은 어두운 진회색으로 설정 */
  color: #2D333B;
}

/* contact 섹션 */
#contact h2 {
  /* 부제목 글자 크기를 32px로 설정 */
  font-size: 32px;
  /* 부제목 글자 색상을 포인트가 되는 파란색으로 설정 */
  color: #3B82F6;
}

/* footer */
footer {
  /* 텍스트를 가운데로 정렬 */
  text-align: center;
  /* 안쪽 여백을 사방으로 24px 설정 */
  padding: 24px;
  /* 배경색을 nav 바와 같은 어두운 진회색으로 설정 */
  background-color: #2D333B;
  /* 글자 색상을 흰색으로 지정 */
  color: white;
  /* 하단 안내 문구이므로 글자 크기를 14px로 작게 설정 */
  font-size: 14px;
}
```

## 반응형 

```css
/* 반응형 — 항상 CSS 파일의 맨 마지막에 위치해야 작동 순서가 꼬이지 않음 */
@media (max-width: 768px) {
  /* 브라우저 가로 폭이 768px 이하(태블릿, 모바일 화면)일 때 적용되는 스타일 */
  
  nav {
    /* 메뉴 항목들을 가로가 아닌 세로 방향으로 정렬 */
    flex-direction: column;

    /* 세로로 쌓인 메뉴 항목들 사이의 간격을 12px로 설정 */
    gap: 12px;

    /* 모바일 화면에 맞춰 메뉴바 안쪽 여백을 16px로 줄임 */
    padding: 16px;
  }

  /* 모바일에서 섹션 여백 줄이기 */
  section {
    /* 모바일 화면이 좁으므로 안쪽 여백을 상하 40px, 좌우 20px로 대폭 줄임 */
    padding: 40px 20px;
  }

  /* 모바일에서 제목 크기 줄이기 */
  #about h1 {
    /* 모바일 화면에서 제목이 너무 크게 차지하지 않도록 글자 크기를 32px로 축소 */
    font-size: 32px;
  }
}
